### Import

To use the component in your project, run the command below into your import statement. This will import the component from your components folder.

```bash
import { Center } from '@gluestack-ui/themed';
```

### Customizing the Center

For customizing the Center, you have to map styled view to Center. You can refer [style.gluestack.io](https://style.gluestack.io/) for more information on how to use styled components.

#### Usage

Default styling of the component can be found in the `components/center` file. For reference, you can view the [source code](https://github.com/gluestack/gluestack-ui/blob/main/example/storybook/src/ui-components/Center) of the styled `Center` component.

```jsx
// import the styles
import { Root } from '../components/core/center/styled-components';

// mapping the Center to root component
const Center = Root;

// using the Center component
export default () => <Center />;
```
